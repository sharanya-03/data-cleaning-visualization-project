# data-cleaning-visualization-project
Data Cleaning and Visualization Project | Python, Pandas, Matplotlib, Seaborn

• Cleaned and analyzed a Superstore sales dataset containing 9,994 records and 21 features.
• Performed exploratory data analysis (EDA) to identify sales trends, profitability patterns, and regional performance.
• Handled data preprocessing tasks including data inspection, duplicate detection, and date transformation.
• Created visualizations such as histograms, bar charts, scatter plots, and correlation heatmaps using Matplotlib and Seaborn.
• Generated business insights through data storytelling and visual reporting.
